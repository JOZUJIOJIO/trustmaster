# Homepage Sanxingdui Bronze Background Design

## Goal

Replace the current abstract cosmic/cloud homepage background with a more mysterious Sanxingdui-inspired visual system using real bronze artifact imagery.

## Visual Direction

Use the classic Sanxingdui bronze mask with protruding eyes as the main low-opacity background signal. Add the bronze sacred tree as a secondary vertical ritual element. The result should feel ancient, unearthed, and ceremonial rather than like a museum poster.

## Assets

- Main artifact: `public/images/sanxingdui-bronze-mask.jpg`
- Secondary artifact: `public/images/sanxingdui-bronze-tree.jpg`
- Source pages:
  - https://commons.wikimedia.org/wiki/File:Ancient_Bronze_Mask_from_Sanxingdui_with_Protruding_Eyes_%26_Ears_a.jpg
  - https://commons.wikimedia.org/wiki/File:Sanxingdui_Bronze_Tree_(9951212965).jpg
- License: CC0 public domain dedication as listed on Wikimedia Commons.

## Homepage Behavior

The existing homepage structure stays intact: nav, theme toggle, language switcher, Taiji visual, hero copy, birth date input, preview card, CTA, footer, and bottom nav. The change is focused on the fixed background layer.

For the cosmic theme, the background uses the real bronze mask centered behind the hero at low opacity with bronze and verdigris grading, plus the sacred tree as a softer side silhouette. For the cloud theme, retain light readability while using a pale bronze/stone treatment of the same assets.

## Constraints

- Keep hero title, input, CTA, and preview card readable on mobile and desktop.
- Avoid a literal full-bright photo background.
- Keep motion subtle and reuse existing animation utilities where possible.
- Include source attribution in code comments and keep asset filenames stable.
